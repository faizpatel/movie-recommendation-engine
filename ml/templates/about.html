About this project:

I'm someone who loves movies, but at the same time am really picky about the ones I watch. After I came accross this dataset containing information relating to 5000 movies, I set out to further develop my understanding of data science, machine learning, and software development principles by building Movie Matcher, a web-based movie recommendation engine.

The first step was to pull the data into Python and do some preprocessing using Pandas and Numpy. My goal was to organize the data so it would be easier to work with later on.

Once I had all my data ready to go, I applied some machine learning principles using the scikit-learn library to peform content based filtering. I created a vector for each of the movies based on keywords in its plot, genre, crew, and cast. I then found the cosine similarity between the vectors of each movie, and the rest of the array, creating a 5000x5000 matrix. By sorting in descending order, I found the movies with the highest similarity to itself for every movie.

After building the model in Python, I was ready to develop the web based application. I used the Flask library on python to connect the ML model front-end created using HTML, CSS, and JavaScript. I also used the TMDB Api to scrape movie posters from the web to make the recommendations page more visually appealing.
